# Statistics-Discrete 

Perl extension for statistical analyses of discrete data.

## Installation

In order to install this module type the following:

```
   $ perl Makefile.PL
   $ make
   $ make test
   $ make install
```
   
## Dependencies

This module does not require other modules and libraries.
